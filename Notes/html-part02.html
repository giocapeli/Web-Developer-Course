HTML Basic - Marking up text

Headings

Títulos

São os headings que atribuem automaticamente certas características de relevância e tamanho para o seu content. São 6 níveis de heading, do <h1> ao <h6>.

Exemplo:

1
2
3
4
5
6 		<h1>Headline 1</h1>
<h2>Headline 2</h2>
<h3>Headline 3</h3>
<h4>Headline 4</h4>
<h5>Headline 5</h5>
<h6>Headline 6</h6>

Resultado visível do exemplo:

Headline 1

Headline 2

Headline 3

Headline 4

Headline 5

Headline 6

Paragraphs

Parágrafos

Os paragraphs contêm o texto de leitura regular da página.

Exemplo:

1
2 		<h1>Este é um paragrafo de texto regular.</h1>
<h2>Este é outro parágrafo.</h2>

Resultado visível do exemplo:

Este é um paragrafo de texto regular.

Este é outro parágrafo.

Strong

Negrito

O strong serve para dar peso ao texto, é um inline element portanto não quebra a linha e pode ser inserido dentro de um texto regular.

Exemplo:

1
2 		<h1>Esté é um <strong>título</strong></h1>
<h2>Esté é um <strong>parágrafo</strong>.</h2>

Resultado visível do exemplo:

Esté é um título

Esté é um parágrafo.

Emphasis

Itálico

O emphasis deixa o texto em itálico.

Exemplo:

1
2 		<h1>Esté é um <em>título</em></h1>
<h2>Esté é um <em>parágrafo</em>.</h2>

Resultado visível do exemplo:

Esté é um título

Esté é um parágrafo.

Lists

Listas

As lists podem ser ordered lists, ordenadas em números, utilizando a tag <ol> e unordered lists, apenas com bullets points, utilizando a tag <ul>. Cada elemento dentro da list deve ser inserido dentro de uma tag <li>

Exemplo:

1
2 		<h1>Esté é um <strong>título</strong></h1>
<h2>Esté é um <strong>parágrafo</strong>.</h2>

Resultado visível do exemplo:

Esté é um título

Esté é um parágrafo.

Images

Inserir Imagens

Usamos a tag image para inserir uma imagem na página. Os seus principais attributes são o src no qual é inserido o caminho da imagem e o alt onde inserimos um texto alternativo para caso a imagem não seja exibida. Ela é um empty element portanto não tem content nem fechamento.

O texto alternativo deve de preferência ser descritiva e não apenas o nome da imagem, assim poderá exercer bem a sua função caso necessário.

Exemplo:

1 <img src="http://startnow.com.br/wp-content/uploads/2017/07/if_new-google-favicon_682665.png">

Anchors

Inserir Links

Usamos a tag anchor para criar um link para uma página ou um arquivo. O seu principal attribute é o href no qual é inserido o caminho do destino, podendo ser um link interno ou externo.

No content colocamos o texto, imagem ou objeto que deverá ser clicado para alcançar o link.

Exemplo:

1
2 		<a href="http://google.com"><img src="http://startnow.com.br/wp-content/uploads/2017/07/if_new-google-favicon_682665.png"></a>><br>
<a href="http://google.com">Link para o google</a>

No content colocamos o texto, imagem ou objeto que deverá ser clicado para alcançar o link.

Resultado visível do exemplo:
