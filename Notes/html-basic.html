O que é HTML?

HTML não é uma linguagem de programação, mas sim uma markup language, ou, linguagem de marcação que é estruturada para dar significado e organizar as informações dos sites. Ela é construída a partir de elementos formados por um conteúdo inserido dentro de uma tag.

O que é um elemento?

O elemento é um conjunto formado pela tag, atributos e conteúdo.

A Opening tag fica entre "<" e ">" (menor e maior) e indica o início do elemento, a Closing tag também fica entre "<" e ">" porém com uma "/" para indicar o fechamento do elemento.

Estrutura básica do HTML:

A estutura básica do HTML é composta por esses elementos:

  1. !DOCTYPE html: A primeira linha é sempre o DOCTYPE, ela não é uma tag mas sim uma instrução ao navegador de que o código a seguir é um html. É obrigatório.
  2. html: É o elemento que envolve todo conteúdo, visível ou não, da página. Também chamado de root element.
  3. head: É o elemento que serve como "container" de todo conteúdo ou instrução que não ficará visível na página principal, como o título da página, keywords, descrição para mecanismos de busca, referências aos CSS, entre outros.
  4. title: Define o título da página que aparece na aba do browser, como padrão ao favoritar a página e como nome da página nas buscas do google.
  5. meta charset="utf-8": Esse elemento indica que o documento utiliza o UTF-8, um padrão de código que inclui a grande maioria dos caracteres conhecidos e utilizados. Problemas na reprodução de acentos podem vir da utilização de outros códigos, como o ASCII.
    6. Outras metatags podem ser usadas. Metatags aparentam não ter nenhum efeito na página em si, mas fornecem informações externas para buscadores por exemplo. Além do charset já visto, segue outras metatags:
    • Author: Nome do autor da página.
    • Copyright: Direitos Autorais.
    • Description: Descrição da página.
    • Expires: Data de expiração do documento.
    • Keywords: Palavras-chave relacionadas à página.
  6. body: É o elemento que armazena o conteúdo que irá ser exibido para o usuário que estiver navegando na página.

Como inserir imagens?

As imagens são inseridas com a tag "img", que é um elemento vazio, ou seja não precisa da tag de fechamento.

A mensagem alternativa deve de preferência ser descritiva e não apenas o nome da imagem, assim poderá exercer sua função.

Marcações:

Os elementos básicos de marcação são:

  • Heading:

    São os cabeçalhos que atribuem automaticamente certas características de relevância, tamanho, para o conteúdo interno. São 6 níveis de headings, do h1 ao h6.
  • Paragraphs:

    Contêm o parágrafo, o texto de leitura regular.
  • Strong:

    Funciona como um "b" (bold), deixa o texto interno em negrito.
  • Emphasis:

    Funciona como um "i" (italic), deixa o texto interno em itálico.
  • Lists:

    Elementos em listas, ordenadas em números (ol) ou não ordenas, com bullet points (ul). Cada item na lista fica dentro de uma tag "li".
  • Divs:

    São containers genéricos para agrupar outros elementos que tem algo em comum ou precisam receber algum atributo em comum. Os divs não são inline, portanto quebram a linha.
  • Spam:

    É um container como o div porém é um elemento inline, que não quebra a linha.

Adicionando comentários:

Os comentáios não são exibidos para o usuário e não tem nenhuma função na página, mas sim servem como guia de leitura e entendimento do código para o desenvolvedor. Os comentários são de suma importância e sua qualidade informativa é proporcional à complexidade e tamanho do código, quanto mais difícil e longo ele é, mais os comentários ajudam tanto quem escreveu quanto quem irá modificar a entender o que foi feito. Os comentários também servem para fazer testes no código, ativando ou desativando elementos.
